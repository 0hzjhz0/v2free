# 『基础算法』前缀和

## 一维前缀和

> 解决多次区间求和问题
>
> 步骤：
>
> 1. 预处理前缀和数组
> 2. 用公式求区间和

```text
si = a1 + a2 + ... + ai
sum(l, r) = a[l] + a[l + 1] + ... + a[r] = s[r] - s[l - 1]
```
